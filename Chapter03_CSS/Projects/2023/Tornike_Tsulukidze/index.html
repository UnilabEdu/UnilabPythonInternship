Typing Methods

Hunt and Peck (two-fingered typing)

ტიპური ბეჭდვის მეთოდი, სადაც რამდენიმე თითს იყენებ ასაკრეფად და უყურებ კლავიატურას.
მახასიათებლები:
  • საშუალო სიჩქარე: 30-40 WPM
  • სიზუსტე: დაბალი, რადგან ინდივიდს ხშირად უწევს კლავიატურაზე შეხედვა.
  • სასწავლად მარტივი, რადგან დამწყებთათვის სტანდარტული მეთოდია.
  • ცუდი პოტენციალი კრეფვის სისწრაფის გასაუმჯობესებლად.
  • ვიზუალურად დამოკიდებული

Touch Typing

ბეჭდვის მეთოდი, რომელიც კრეფვის სისწრაფეს მნიშვნელოვნად ზრდის.
მახასიათებლები:
  • საშუალო სიჩქარე: 40-80 WPM
  • პოტენციალი მეთოდის დახელოვნების შემდეგ: 100+ WPM
  • სიზუსტე: მაღალი, რადგან ინდივიდს მხოლოდ ეკრანზე უწევს შეხედვა.
  • დასაწყისში სასწავლად შედარებით ძნელი, რადგან საჭიროა კუნთის მეხსიერების დახვეწა.
  • მოსახერხებელი განლაგება, ნაკლები თითების მოძრაობა
  • ვიზუალურად დამოუკიდებელი

Hybrid Typing

ბეჭდვის მეთოდი, სადაც გაერთიანებულია Hunt and Peck-ისა და Touch Typing-ის მახასიათებლები.
მახასიათებლები:
  • საშუალო სიჩქარე: 30-70 WPM
  • მოსახერხებელი, რადგან კრეფვა დამოკიდებულია სრულიად კუნთის მეხსიერებაზე
  • განვითარების დაბალი პოტენციალი, სანამ Touch Typing უნარების განვითარებას არ შეეცდები..
  • ნაწილობრივ ვიზუალურად დამოკიდებული.

Thumb Typing

ბეჭდვის მეთოდი ტელეფონის მომხმარებლებისთვის.
მახასიათებლები:
  • საშუალო სიჩქარე: 20-40 WPM
  • მოსახერხებელი ტელეფონის მომხმარებლებისთვის შეზღუდული კლავიატურის სივრცით.
  • სასწავლად მარტივი ძირითადი გამოყენებისთვის, თუმცა გამომწვევი სიჩქარის გაზრდისთვის.
  • ვიზუალურად დამოკიდებული, თუმცა პატარა ეკრანთან და ვირტუალურ კლავიატურასთან ურთიერთობისას არც ისე მნიშვნელოვანია.

Dvorak Simplified Keyboard

ბეჭდვის მეთოდი, რომელიც გათვლილია კრეფის სიჩქარისთვისა და ნაკლები თითის მოძრაობისთვის.
მახასიათებლები:
  • საშუალო სიჩქარე: 50-80 WPM
  • აქვს პოტენციალი უფრო სწრაფად აკრეფვის, ვიდრე Touch Typing-ს
  • სასწავლად ძNელი, რადგან საჭიროა ადაპტირება ახალ წყობასთან, QWERTY-იდან DVORAK-ზე გადართვა.
  • მოსახერხებელი ხელის ნაკლები მოძრაობით.
  • იშვიათად გამოყენებული, თუმცა DVORAK-ის მომხმარებლების გადმოცემით გაცილებით სწრაფი და კომფორტული.

მე ვიყენებ Touch Typing-ს, ფოტოზე გამოსახულია საკუთარი გამოცდილება Typing ინდუსტრიაში.
